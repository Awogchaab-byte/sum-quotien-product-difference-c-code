# ➕ Basic Arithmetic Operations in C++

This program takes **two integer values** from the user and calculates their **sum**, **difference**, **product**, and **quotient** using standard arithmetic operators.

---

## 🧠 Objective
To practice basic input/output and arithmetic operations in C++.

---

## 📜 Problem Statement
> Write a C++ program that takes in two integer values from the user and calculates the sum, difference, product, and quotient using the appropriate operators.

---

## 💻 Code Overview
The program:
1. Prompts the user to enter two integers.
2. Performs the following operations:
   - Addition (`+`)
   - Subtraction (`-`)
   - Multiplication (`*`)
   - Division (`/`)
3. Displays the results neatly.
